A simple, efficient, and user-friendly Task Management Application built with React.js. TaskMan helps users create, track, complete, and manage tasks with persistent storage and a visual progress tracker.

ğŸš€ Features
1. Task Input Form

Users can add tasks using an input field and submit button.

Input validation ensures tasks are not empty or duplicated.

2. Task List

Displays tasks dynamically with real-time updates.

Each task shows its completion status.

Users can mark tasks as complete or delete them.

3. Persistent Data

Tasks are saved using localStorage.

All data remains intact even after refreshing or closing the browser.

4. Progress Tracker

Shows the percentage of completed tasks through a visual progress bar.

Progress updates automatically based on user actions.

5. Task History

Stores completed tasks in a history section.

Users can restore tasks back to the active list or permanently delete them.
